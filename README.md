# Installation Guide
requirements:
- Node
- Mongodb

then:
- Rename env files from .env.example to .env on server and public directory
- install the dependencies
  - cd server
  - yarn
  - cd ..
  - cd public
  - yarn

for frontend:
- Type this on your terminal to run:
  - cd public
  - yarn start

for backend:
- Type this on your terminal to run:
  - cd server
  - yarn start
